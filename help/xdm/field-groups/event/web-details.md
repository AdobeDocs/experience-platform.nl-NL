# [!UICONTROL Web Details] schemaveldgroep

>[!NOTE]
>
>De namen van verschillende groepen schemavelden zijn gewijzigd. Document weergeven op [veldgroepnaapupdates](../name-updates.md) voor meer informatie .

[!UICONTROL Web Details] is een standaardschemagebiedgroep voor [[!DNL XDM ExperienceEvent] class](../../classes/experienceevent.md), gebruikt om informatie te beschrijven over gebeurtenissen met betrekking tot webdetails zoals interactie, paginadetails en referentie.

![](../../images/field-groups/web-details.png)

| Eigenschap | Gegevenstype | Beschrijving |
| --- | --- | --- |
| `web` | [Webinformatie](../../data-types/web-information.md) | Beschrijft verbindingskliks, Web-pagina details, verwijzingsinformatie, en browser details. |
