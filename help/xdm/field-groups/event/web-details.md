# [!UICONTROL Web Details] schemaveldgroep

>[!NOTE]
>
>De namen van verschillende groepen schemavelden zijn gewijzigd. Zie het document op [updates van de gebiedsgroepnaam](../name-updates.md) voor meer informatie.

[!UICONTROL Web Details] is een standaardschemagebiedgroep voor de  [[!DNL XDM ExperienceEvent] klasse](../../classes/experienceevent.md), die wordt gebruikt om informatie betreffende Web detailgebeurtenissen zoals interactie, paginadetails, en verwijzer te beschrijven.

![](../../images/field-groups/web-details.png)

| Eigenschap | Gegevenstype | Beschrijving |
| --- | --- | --- |
| `web` | [Webinformatie](../../data-types/web-information.md) | Beschrijft verbindingskliks, Web-pagina details, verwijzingsinformatie, en browser details. |

{style=&quot;table-layout:auto&quot;}

* [Voorbeeld van vulling](https://github.com/adobe/xdm/blob/master/components/mixins/experience-event/experienceevent-web.example.1.json)
* [Volledig schema](https://github.com/adobe/xdm/blob/master/components/mixins/experience-event/experienceevent-web.schema.json)
