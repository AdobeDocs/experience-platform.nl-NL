# Samenvoegen-id opnieuw instellen

>[!IMPORTANT]
>
>Deze handeling is vervangen. Het gebruik [&#x200B; verzamelt interne verbinding klikt &#x200B;](../configure/data-collection.md#collect-internal-link-clicks) montages in plaats daarvan.

Met het actietype **[!UICONTROL Reset merge ID]** kunt u gebeurtenissen scheiden die op dezelfde pagina worden aangeroepen. Het wordt doorgaans gebruikt in internecontrolescenario&#39;s waar u meerdere ladingen kunt hebben die u naar Adobe wilt verzenden. Met deze actie kunt u de samenvoegings-id van een gebeurtenis opnieuw instellen, zodat deze na aankomst bij de Edge Network niet als onderdeel van dezelfde gebeurtenis worden beschouwd.

Als u wilt controleren hoe de veelvoudige gebeurtenissen op de zelfde pagina worden gescheiden of samengevoegd, gebruik [&#x200B; verzamel interne verbinding klikt &#x200B;](../configure/data-collection.md#collect-internal-link-clicks) optie wanneer het vormen van de markeringsuitbreiding.
